Виборы Опен Сорс. версия 0.2
============

Выборы с доверием к ЦВК в один тур без подтверждения.


1. Основные принципы. 

1.1. Бюлетень для голосования представляет из себя цифровой код на бланке строгой отчётности, запечатанный в конверте. Код состоит из открытого ключа для голсования (и опционально некоторого кода, для подтверждения подлинности бюлетеня... но зачем ведь всеравно код на бланке строгой отчётности). ВК печатает бюлетени на основе закрытого ключа, который остаётся тайной до конца голосования. Электронная копия проголосовавшего бюлетеня передаётся в ВК, бумажная копия кидается в урну для голосования для наблюдателей. После окончания голосования наблюдатели сканируют все бюлетени и публикуют их. Для подтверждения достоверности должны совпадать зашифрованные данные наблюдателей и ВК. После окончания выборов и публикации/проверки шифрованных бюлетеней ВК публикует закрытый ключ. На основе закрытого ключа система дешифрует данные ВК и наблюдателей, затем система считает результаты. 

1.2. Терминал для голосования предстваляет собой планшетный компьютер со сканером/принтером специальной модели. В кабинке для голосования избиратель вскрывает конверт, достаёт бюлетень для голосования и вставляет его в терминал для голосования. Терминал сканирует код для голосования и подтверждает подлинность бюлетеня после чего избиратель выьирает за что он голосует, затем терминал печатает зашифрованный выбор на бюлетене.  Зашифрованный выбор сохраняется на диске терминала или передаётся онлайн на сервер ВК. Бумажный бюлетень избиратель сворачивает вдвое и кидает в урну для голосования.
